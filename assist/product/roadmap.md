# Roadmap

## Phase 0: Foundation Planning

Goals:

- Define architecture.
- Define tenant model.
- Define module boundaries.
- Define industry model.
- Define agent rules.
- Define versioning and release process.

Status:

- Started with the initial `assist/` pack.

## Phase 1: Platform Core

Goals:

- Tenant management.
- Authentication.
- Roles and permissions.
- Subscription and activation.
- Feature flags.
- Central design system.
- App shell.
- Audit trail.
- Settings.
- Basic queue system.
- Basic event system.

## Phase 2: Business Core

Goals:

- Customer and vendor masters.
- Item and inventory foundation.
- Billing foundation.
- Accounting foundation.
- Tasks and activities.
- Mail and notifications.
- Reporting foundation.

## Phase 3: Industry Packs

Goals:

- POS billing pack.
- Garments billing pack.
- Garments manufacturing pack.
- uPVC pack.
- Offset printing pack.
- Software services pack.

## Phase 4: Offline And Desktop

Goals:

- Electron app shell.
- Local tenant store.
- Offline sync manager.
- Conflict workflow.
- Offline POS or billing workflow.
- Sync monitoring.

## Phase 5: Mobile

Goals:

- Expo mobile app.
- Mobile authentication.
- Mobile dashboards.
- Tasks and activity.
- Customer lookup.
- Basic sales or billing flows where needed.
- Mobile sync strategy.

## Phase 6: Integrations

Goals:

- WhatsApp.
- Telegram.
- Mail providers.
- e-Invoice.
- e-Way bill.
- External APIs.
- Import and export jobs.

## Phase 7: AI Assistants

Goals:

- CODEIT development workspace.
- ZERO business assistant.
- Permission-aware business tools.
- AI audit trail.
- Model routing.
- Business prediction workflows.

## Phase 8: Scale And Enterprise

Goals:

- Advanced tenant deployment options.
- Dedicated containers for selected tenants.
- Advanced reporting.
- Backup and restore automation.
- Support console.
- Monitoring and alerting.
